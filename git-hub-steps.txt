--- First Time ---
1. cd into that folder
2. git config --global user.email "<your email>"
3. git init
4. git add --all
5. git commit -m "1"
6. Use the link of that repository - the link that is the remote link to that repository
7. Copy that link to run in command line
8. git push origin master

--- Update ---
1. cd into that folder
2. git add --all
3. git commit -m "<next number>"
4. git push origin master
